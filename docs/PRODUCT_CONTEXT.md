# CampaignOps Product Context

## Identity

Project name: CampaignOps

CampaignOps is an AI-assisted event and campaign operations platform. It is broader than a dashboard, AI wrapper, generic event app, CRM, chatbot, or task manager.

It is being built for real internal usage by an event/campaign/startup team working across:

- Government events
- CSR campaigns
- Community events
- Cafe events
- Outreach campaigns
- Client-facing execution work

## Core Vision

CampaignOps converts a raw event or campaign idea into a complete execution workspace.

The system should support:

- Event/campaign intake
- Event type classification
- Scale and budget planning
- AI-generated execution plan
- Task assignment
- Vendor/contact management
- Stakeholder/outreach management
- Content calendar planning
- Budget tracking
- File/document storage
- Activity logs
- Reports
- Post-event debriefs
- Reusable playbook/knowledge base

The long-term vision is to make every completed event improve the system's future intelligence.

Every event should create reusable knowledge:

- What was planned
- What actually happened
- What budget was estimated
- What budget was actually used
- Which vendors worked well
- Which outreach messages worked
- What problems occurred
- What should be remembered next time
