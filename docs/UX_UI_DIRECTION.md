# CampaignOps UX/UI Direction

## Feel

The UI should feel like:

- Premium Himalayan event operations command centre

It should not feel like:

- Generic SaaS dashboard
- Government portal
- Cartoon event app
- Cyberpunk AI app
- Over-decorated landing page

## Visual Mood

- Premium command centre
- Himalayan / Uttarakhand-inspired
- Cinematic event operations
- Large public gatherings
- Ministers addressing crowds
- Stage lights
- Audience silhouettes
- Media cameras
- Barricades
- Tents
- Banners
- Mountain haze
- Warm evening tones
- Clean dashboard density

## Color Palette

- Deep Ink: `#0D0D0D`
- Himalayan Cream: `#F5F2ED`
- Dhak Orange: `#E8602C`
- Forest Green: `#2A6B4F`
- Brass Gold: `#C4A85A`
- Mist Grey: `#E7E2D8`
- Glacier Blue: `#7F9DB9`

Usage:

- Use Dhak Orange for primary CTAs and active states.
- Use Forest Green for completed/success.
- Use Brass Gold for premium/government/high-priority accents.

## Layout

- Desktop-first
- Left sidebar
- Topbar
- Main content area
- Event cards
- Metric cards
- Event command centre tabs
- One event equals one command centre
- Avoid deep nesting
